##Install virtual box

	sudo apt-get install virtualbox

##Install vagrant
- Download [Vagrant deb package](https://www.vagrantup.com/downloads.html):

	    cd /tmp/mozilla_ben0
	    dpkg -i vagrant_1.9.5_x86_64.deb

- Clone your vagrant repo and install it!
